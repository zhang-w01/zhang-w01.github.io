# MarkText笔记使用教程

## 快捷键

| ctrl+j       | 调出侧边栏           |
|:------------:|:---------------:|
| ctrl+t       | 新建一个笔记本         |
| ctrl+shift+k | 创建代码区           |
| ctrl+shift+t | 插入列表            |
| ctrl+shift+i | 插入图片            |
| Ctrl+Shift+H | <mark>高亮</mark> |
| Ctrl+`       | `标签`            |
| Ctrl+Shift+M | 数学公式$y=x^2$     |
| Ctrl+U       | <u>文字下划线</u>    |
| Ctrl+B       | **文字粗体**        |
| Ctrl+L       | [](连接)          |
| Ctrl+D       | ~~文字删除线~~       |
| Ctrl+Alt+H   | 插入HTML块         |
| Ctrl+Shift+Q | 引用块             |
| Ctrl+Alt+X   | todo列表          |

## 快捷符号

| #   | 标题        |
|:---:| --------- |
| -   | ·项目列表     |
| 1.  | 1.有序列表    |
| *   | *倾斜字体*    |
| **  | **加粗字体**  |
| `   | `文字标签`    |
| >   | 引用块       |
| --- | 注释        |
| ~~  | ~~文字删除线~~ |
